# digital.kotor.me

Digitalna platforma Opštine Kotor za upravljanje konkursima, tenderima i online plaćanjima.

## O projektu

Ova Laravel aplikacija pruža digitalne usluge građanima i privrednicima Kotora:

- Online plaćanje opštinskih prihoda
- Portal za konkurse (žensko i omladinsko preduzetništvo)
- Sistem za tendersku dokumentaciju
- Upravljanje prijavama i evaluacijom
- Generisanje ugovora i izvještaja

## Tehnologije

- Laravel Framework
- PHP 8.3+
- MySQL/PostgreSQL
- Tailwind CSS
- Vite

## Instalacija

```bash
composer install
npm install
cp .env.example .env
php artisan key:generate
php artisan migrate
npm run dev
```
